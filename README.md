# Binazy Official Website Frontend

This project is the frontend of the **Binazy Official Website**, built using **React + Vite**. It is a modern, responsive, and scalable web application designed to showcase company services, industries, and contact information with a clean user interface.

---

## Tech Stack

- React.js
- Vite
- React Router DOM
- Tailwind CSS
- JavaScript (ES6+)

---

## Features

- Modern responsive UI
- Fast performance with Vite
- Reusable React components
- Dynamic routing
- Services pages with data-driven structure
- Industries pages with dynamic content
- Contact page with form validation
- FAQ section
- Mobile-friendly navigation
- Clean folder structure
